4. Let‚Äôs create the input file üß™#


This inp. file is gonna be the base for the creation of our .orcacosmo that we are gonna use in the openCOSMO-RS program.

The first step is to generate a .inp type file with some information about the molecule. The formatting should be the following:

name_of_molecule [TAB] SMILES_of_molecule [TAB] xyz_file [TAB] charge_of_molecule [TAB] geometry_optimization

  • name_of_molecule: any name like methanol, water, etc.

  • SMILES_of_molecule: SMILES are way to represent a molecule‚Äôs structure through string characters. For methanol it would be: CO, for water O. In websites like Pubchem and Chemspider SMILES are available for a variety of molecules

  • xyz_file: text file used to describe the geometry of the molecule, it contains the 3D information about all the atoms in the molecule

  • charge_of_molecule: can be neutral molecules: 0, or cations +1, +2 or anions: -1, -2

  • geometry_optimization: if the optimization should be performed, write TRUE, if not, FALSE or leave blank. The geometry optimization is to find the arrangement of atoms in a molecule that corresponds to the lowest possible energy state, so called equilibrium geometry

The xyz_file is optional to include as well as the geometry_optimization

  • If we do not have an xyz_file, the program will use Python‚Äôs library RDKit or Balloon to search for a conformer before sending it to xtb to do an extensive conformer search. At the end the program will give us an xyz file that generated based on those calculations

  • If we do not include the geometry optimization, the last calculations will be made using the xyz file only without optimizing the position of the atoms


If we want to create an input file for the water molecule we can do:

water [TAB] O [TAB] [TAB] 0

In this we did not include the xyz_file and the geometry_optimization


For every line that you have the following symbols: ‚ö†Ô∏è‚ö†Ô∏è‚ö†Ô∏è you need to modify and specify the correct address in your computer

# Let's now create and save our file
#the [TAB] in python is represented with '\t'
content = 'water\tO\t\t0'

#set up the file path and name the file ‚ö†Ô∏è‚ö†Ô∏è‚ö†Ô∏è
file_path = r"INPUT_FILES\water.inp"

# Open a file in write mode
with open(file_path, "w") as file:
    file.write(content)

print(f"File created  at {file_path}")

Let's do it for butane and pentane:

  • For butane we will include the xyz_file and the geometry_optimization

  • For pentane we will omit both geometry_optimization



First let‚Äôs create the xyz_file file for butane, an xyz_file file is a simple text file format used to describe atomic coordinates in a molecule. It can be obtained from online databases.

This is what the file should have:

  1. First Line: Contains the number of atoms in the molecule.

  2. Second Line: A comment line, which can be left blank or used to describe the molecule.

  3. Subsequent Lines: Each line corresponds to an atom and includes the chemical symbol of the atom and the X, Y, and Z coordinates of the atom in space.
